# Project EDGE

> Fast. Lethal. Cinematic.

Project EDGE é um projeto de adaptação das regras de Cyberpunk RED com foco em simplificação mecânica, acessibilidade e melhoria da experiência do jogador, preservando a identidade do cenário e a letalidade característica do sistema.

---

## Objetivos

- Simplificar mecânicas complexas.
- Reduzir o número de Skills.
- Diminuir o tempo gasto consultando regras.
- Tornar o sistema mais acessível para novos jogadores.
- Manter a sensação de perigo e letalidade de Cyberpunk RED.
- Integrar melhorias inspiradas no Edgerunners Mission Kit.
- Produzir uma Wiki para Obsidian.
- Desenvolver um Livro do Jogador.
- Desenvolver um Livro do Mestre.

---

## Filosofia de Design

Project EDGE é guiado por cinco princípios:

- **Fast** — regras rápidas de aprender e consultar.
- **Lethal** — combate perigoso e consequências reais.
- **Cinematic** — incentivar cenas memoráveis em vez de microgerenciamento.
- **Accessible** — fácil para jogadores iniciantes.
- **Narrative First** — a narrativa sempre vem antes da regra.
