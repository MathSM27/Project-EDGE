# Project EDGE

> Fast. Lethal. Cinematic.

Project EDGE é um projeto de adaptação das regras de Cyberpunk RED com foco em simplificação mecânica, acessibilidade e melhoria da experiência do jogador.

## Objetivos

- Simplificar regras complexas.
- Melhorar o fluxo de jogo.
- Preservar a identidade de Cyberpunk RED.
- Criar uma Wiki para Obsidian.
- Desenvolver um Livro do Jogador e um Livro do Mestre.

## Estado do Projeto

Em desenvolvimento

Versão atual: **v0.1.0**
